# Future Fly Global Ventures - Travel Agency Website

Welcome to the Future Fly Global Ventures travel agency website built with Next.js and Tailwind CSS.

## Features

- **Modern Hero Section**: Dynamic animations with Framer Motion and video-like background.
- **Detailed Service Offerings**: Clean, icon-based service descriptions.
- **Contact Information**: Easy-to-find contact details including phone, email, and address.
- **Responsive Design**: Fully optimized for mobile and desktop devices.
- **Theme**: Dark mode, glassmorphism, and premium aesthetics.

## Getting Started

1.  **Install Dependencies**:
    Make sure you have installed the required packages:

    ```bash
    npm install lucide-react framer-motion
    ```

2.  **Run Development Server**:

    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```

3.  **Open in Browser**:
    Visit [http://localhost:3000](http://localhost:3000) to see the website.

## Tech Stack

-   **Next.js 15+** (App Router)
-   **React**
-   **Tailwind CSS v4**
-   **Framer Motion** (Animations)
-   **Lucide React** (Icons)

## Customization

You can customize the content in `app/page.tsx`. Images used are from Unsplash and configured in `next.config.ts`.
